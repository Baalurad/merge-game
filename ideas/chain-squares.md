# Цепочка: Квадраты → Корона

**Статус:** 💬 Концепт

## Концепт

Идея жены: квадратные камни сливаются, растут, и в итоге образуют украшения с прямоугольными камнями.  
Форма — **emerald-cut** (прямоугольная ступенчатая огранка) — держится на протяжении всей цепочки как визуальный якорь.

В отличие от рубина (камень → разные категории украшений), тут категории тоже меняются,  
но форма камня остаётся одинаковой — всегда прямоугольник с ровными гранями.

**Исправление оригинала:** в эскизе жены L1 — куча обломков. Это создаёт регресс (много → один).  
Заменили на: L1 — один маленький сырой кусочек, эволюция идёт вперёд.

| # | Объект | Как выглядит |
|---|---|---|
| 1 | Сырой осколок | Один неровный матовый прямоугольный кусок |
| 2 | Гладкий прямоугольник | Полированный, без огранки, один блик сверху |
| 3 | Emerald-cut камень | Ступенчатая огранка, первые чёткие грани |
| 4 | Крупный камень | Тот же стиль, заметно больше |
| 5 | Два камня | Симметричная пара рядом |
| 6 | Кольцо с камнем | Серебряное кольцо с прямоугольным аметистом |
| 7 | Браслет | Ряд прямоугольных камней в серебре, вид сверху |
| 8 | Ожерелье | Центральный крупный + ряд по дуге |
| 9 | Корона | Прямоугольные аметисты вдоль обода |
| 10 | Имперская корона | Высокая, богатая, серебро, много прямоугольных камней |

**Визуальное разграничение с рубиновой короной (L9–L10):**  
рубин = золото + круглые камни, аметист = серебро + прямоугольные камни.
