## Цепочки элементов

| Цепочка | Цвет | Статус | Описание |
|---|---|---|---|
| [Рубин](chain-ruby.md) | Красный + золото | ✅ Готово | Осколок → огранка → кольцо → серёжки → колье → корона |
| [Капли → Кристалл](chain-drops.md) | Синий (сапфир) | ✅ Готово | Мутная капля трансформируется в огранённый кристалл, спрайты в `assets/crystal-cartoon/` |
| [Квадраты → Корона](chain-squares.md) | Фиолетовый (аметист) | 💬 Концепт | Сырые прямоугольные камни через украшения к имперской короне |

**Стадии:** 💬 Концепт → ✏️ Промпты → 🎨 Генерация → ✅ Готово

> Концепт включает обязательно: описание 10 уровней + дизайн корзинки (источник ресурсов).
