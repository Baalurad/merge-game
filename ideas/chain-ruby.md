# Цепочка: Рубин

**Цвет:** красный + золото  
**Статус:** ✅ Готово

---

## Концепт

Необработанный рубин превращается в роскошное ювелирное украшение.  
Логика: **категория объекта меняется** — сначала сам камень (осколок → огранка), потом украшения (кольцо → серёжки → колье → корона).

| # | Объект | Как выглядит |
|---|---|---|
| 1 | Осколок | Маленький, сырой, матовый, неровная форма |
| 2 | Кабошон | Полированный гладкий купол, без огранки |
| 3 | Огранённый рубин | Brilliant-cut, множество граней, сильные блики |
| 4 | Простое кольцо | Тонкое золотое кольцо с одним рубином |
| 5 | Перстень-печатка | Широкое золотое кольцо с большим рубином и гравировкой |
| 6 | Гвоздик | Золотая серёжка-гвоздик с рубином |
| 7 | Подвеска | Серёжка — рубиновая капля на дужке |
| 8 | Колье | Золотое колье с рубиновой подвеской |
| 9 | Тиара | Изящная тиара с рубинами |
| 10 | Корона | Королевская корона, золото, рубины и бриллианты |

## Промпты

Подробные промпты и итерации по каждому уровню: [`scripts/ruby-chain-prompts.md`](../scripts/ruby-chain-prompts.md)

**Ключевые находки:**
- `front-facing view` / `three-quarter view angle` — обязателен явный вид
- `ONLY the necklace`, `isolated single item` — запрет лишнего
- `brilliant`, `luxury`, `opulent` — добавляют детали и блеск
- Размер предмета: `itemSize` растёт 88px (L1) → 188px (L10)
