# Фича: Панель заказчиков (Customer Panel)

**Статус:** 🔧 Базовая реализация есть, нужна доработка

- До 4 заказчиков одновременно
- 1 товар на заказ

## 1. Горизонтальный свайп (>4 заказчиков)

**Проблема:** Phaser не имеет встроенного горизонтального scroll-контейнера.

**Реализация:**
- Создать `Phaser.GameObjects.Container` для панели заказчиков
- Отслеживать `pointerdown` / `pointermove` / `pointerup` на панели
- Сдвигать контейнер по X, ограничить `clampX` по ширине контента
- Добавить momentum (затухающий скролл после отпускания)

**UX:**
- Видимая область: 4 заказчика (или 3.5 для намёка на скролл)
- Показывать индикатор-точки или мини-скроллбар под панелью
- Макс. заказчиков одновременно: предлагается **8**

## 2. Мульти-товарные заказы

**Текущая модель:** `{ type, level, reward, expiresAt }`

**Новая модель:**
```js
{
  customerId,
  items: [ { type, level }, { type, level }, ... ],  // 1–3 предмета
  reward,      // сумма за каждый предмет + бонус за комплект
  expiresAt,
}
```

**Логика выполнения:**
- Заказ считается выполненным только когда **все** items поставлены
- Промежуточный прогресс: отмечать выполненные items галочкой прямо в карточке
- Бонус за комплект: +20% к reward если ≥2 items одновременно

**Сложность заказа по прогрессу:**
- Первые 5 минут: только 1-item заказы
- После 5 мин: появляются 2-item
- После 15 мин: могут появляться 3-item

## 3. Спрайты заказчиков

**Концепт:** мультяшные женские персонажи в ювелирном стиле (под Светлану).

**Типы персонажей (предлагается ~6 штук):**
| ID | Описание | Настроение |
|---|---|---|
| lady_classic | Классическая дама в шляпке | Элегантная |
| girl_trendy | Молодая модница | Восторженная |
| princess | Принцесса с короной | Капризная |
| countess | Графиня в вечернем платье | Взыскательная |
| witch | Ведьма (юмористичная) | Таинственная |
| fairy | Фея с крылышками | Добродушная |

**Формат:**
- Размер карточки заказчика: ~130×160px (portrait)
- Спрайт персонажа: 80×100px внутри карточки
- Генерация: Replicate `recraft-ai/recraft-v3`, `digital_illustration`
- Промпты: добавить в новый файл `scripts/gen-customers.js`

**Анимации состояний:**
- Idle: лёгкое покачивание (Phaser tween, -3°/+3° rotation)
- Waiting: нетерпеливое постукивание (анимированный спрайт или tween)
- Happy (заказ выполнен): прыжок + звёздочки
- Angry (заказ истёк): красное лицо + восклицательный знак

## 4. Анимация выполнения заказа

**Вариант A — Простой (рекомендуется для MVP):**
- Элемент летит дугой от ячейки к карточке заказчика (Phaser tween по кривой Безье)
- При достижении: вспышка + particle burst (звёздочки цвета элемента)
- Карточка заказчика: scale bounce (0.9 → 1.15 → 1.0)
- Счёт: +N цифра всплывает и исчезает вверх

**Вариант B — Полный:**
- Всё из варианта A
- Персонаж переходит в состояние Happy (анимация)
- Если последний item в заказе — карточка уезжает вправо и исчезает

**Реализация (Phaser):**
```js
// Дуга полёта
this.tweens.add({
  targets: element,
  x: targetX, y: targetY,
  ease: 'Cubic.easeIn',
  duration: 400,
  onComplete: () => { spawnParticles(); scorePopup(); }
});
```
